Svgbob-pro is an authoring tool to make diagrams out of plain text.
Plain text base diagrams are used in places where only plain text is the viable options,
such as source code.


       .---.
      /-o-/--
   .-/ / /->
  ( *  \/
   '-.  \
      \ /
       '
It uses a combination of characters
which are readily available on your keyboards.
